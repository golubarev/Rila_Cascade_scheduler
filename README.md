# Rila_Cascade_scheduler

This project aims to create an automation for easier preparation of the power production schedules of Rila Cascade.

# Rila Cascade is an HPP Cascade:
1. DAM Kalin and DAM Karagyol
2. PSHPP Kalin
3. RESERVOIR Kamenitsa
4. HPP Kamenitsa
5. RESERVOIR Pastra
6. HPP Pastra
7. RESERVOIR Rila
8. HPP Rila
